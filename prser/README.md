# 🤖 Telegram Parser Bot

Бот для парсинга сообщений из Telegram-чатов с пересылкой в избранное.

## 🚀 Деплой на Railway

1. Форкните этот репозиторий
2. Создайте проект на Railway
3. Подключите GitHub репозиторий
4. Добавьте переменные окружения
5. Настройте Volume для хранения данных

## 📝 Переменные окружения

Обязательные:
- `BOT_TOKEN` - токен бота от @BotFather
- `ALLOWED_USERS` - ID пользователей через запятую
- `ACCOUNT_1_NAME` - имя аккаунта
- `ACCOUNT_1_API_ID` - API ID из my.telegram.org
- `ACCOUNT_1_API_HASH` - API Hash
- `ACCOUNT_1_PHONE` - номер телефона

Опциональные:
- `DEFAULT_LIMIT` - лимит сообщений (по умолчанию 50)
- `MAX_FORWARD_PER_RUN` - максимум пересылок (по умолчанию 30)
- `CHECK_INTERVAL_HOURS` - период проверки в часах (по умолчанию 24)
- `MIN_TEXT_LENGTH` - минимальная длина текста (по умолчанию 5)

## 📊 Хранение данных

Данные сохраняются в папке `data/`, которая монтируется как Volume для постоянного хранения.
